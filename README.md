# 🎬 Subtitle Translator (Powered by Gemini)

این پروژه یک ابزار هوشمند برای ترجمه خودکار فایل‌های زیرنویس (`.srt` و `.vtt`) از انگلیسی به فارسی است. با استفاده از مدل **Gemini Flash** گوگل، ترجمه‌ها بسیار روان، طبیعی و با حفظ ساختار زمانی فایل زیرنویس انجام می‌شوند.

## ✨ ویژگی‌ها
* **ترجمه هوشمند:** فراتر از ترجمه تحت‌اللفظی، با درک اصطلاحات و فرهنگ زبان فارسی.
* **حفظ ساختار:** شماره خطوط و زمان‌بندی‌ها (Timestamp) بدون تغییر باقی می‌مانند.
* **پردازش دسته‌ای (Batch):** برای مدیریت محدودیت‌های API و سرعت بیشتر، زیرنویس‌ها در دسته‌های ۴۰۰ خطی پردازش می‌شوند.
* **پشتیبانی از فرمت‌ها:** قابلیت کار با هر دو فرمت رایج `SRT` و `VTT`.
* **ساختار پوشه‌ای:** حفظ ساختار پوشه‌های ورودی در پوشه خروجی.
